# TriLLM Arena Deployment Guide

## Quick Deploy to Streamlit Cloud (FREE, PUBLIC, FOREVER)

### Step 1: Create GitHub Repository

```bash
cd "/Users/soumyadarshandash/debate ai"
git init
git add .
git commit -m "Initial commit: TriLLM Arena v2.0"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/trillm-arena.git
git push -u origin main
```

### Step 2: Sign Up for Streamlit Cloud

1. Go to: https://streamlit.io/cloud
2. Sign up with GitHub
3. Click "New app"
4. Select:
   - Repository: `your-username/trillm-arena`
   - Branch: `main`
   - Main file path: `streamlit_app.py`

### Step 3: Deploy

Streamlit Cloud will automatically deploy and give you a public URL:
```
https://your-app-name.streamlit.app
```

### Your App is Now PUBLIC & FOREVER ONLINE!

Anyone can access: `https://your-app-name.streamlit.app`
